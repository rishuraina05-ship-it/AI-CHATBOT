# AI-CHATBOT 🤖
AI-powered chatbot built using Python, Streamlit, and LLM API to generate real-time conversational responses with context memory.

## Overview
This is an AI-powered chatbot that generates intelligent and context-aware responses using a Large Language Model API.

## Features
- Real-time conversational chatbot
- Context memory using session state
- Clean UI using Streamlit
- Fast AI responses via API

## Tech Stack
- Python
- Streamlit
- LLM API (Groq/OpenAI)
- Session State Management

## How to Run
1. Install dependencies:
   pip install streamlit groq

2. Run the app:
   python -m streamlit run app.py

## Use Case
This project demonstrates building real-world AI applications using modern LLMs and deploying them with a simple UI.
